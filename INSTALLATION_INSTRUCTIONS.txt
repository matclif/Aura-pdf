========================================
           WINDOWS INSTALLATION
========================================

Option 1: Easy Installation (Recommended)
----------------------------------------
1. Double-click "install-windows.bat"
2. Select option 1 (Install with Setup)
3. Follow the installation wizard
4. Aura PDF™ will be installed in your Programs folder
5. You can find it in your Start Menu

Option 2: Manual Installation
-----------------------------
1. Run "Aura PDF™ Setup 2.0.0.exe" from the dist folder
2. Follow the installation wizard
3. Choose installation directory (default: C:\Program Files\Aura PDF™)
4. Create desktop shortcut (optional)
5. Complete installation

Option 3: Portable Version
-------------------------
1. Run "Aura PDF™ 2.0.0.exe" from the dist folder
2. No installation required - runs directly
3. You can copy this file anywhere and run it

========================================
            MAC INSTALLATION
========================================

Option 1: Intel Mac
------------------
1. Double-click "Aura PDF™-2.0.0.dmg"
2. Drag "Aura PDF™" to your Applications folder
3. Launch from Applications or Launchpad

Option 2: Apple Silicon Mac (M1/M2/M3)
-------------------------------------
1. Double-click "Aura PDF™-2.0.0-arm64.dmg"
2. Drag "Aura PDF™" to your Applications folder
3. Launch from Applications or Launchpad

Note: If you get a security warning, go to:
System Preferences > Security & Privacy > General
Click "Open Anyway" next to the Aura PDF™ message

========================================
            TROUBLESHOOTING
========================================

Windows Issues:
- If installation fails, try running as Administrator
- Make sure you have enough disk space
- Check Windows Defender isn't blocking the installer

Mac Issues:
- If app won't open, right-click and select "Open"
- Go to System Preferences > Security & Privacy > General
- Click "Open Anyway" if prompted

General Issues:
- Make sure you have the latest version of your operating system
- Restart your computer if the app doesn't launch
- Check that you have sufficient RAM available
